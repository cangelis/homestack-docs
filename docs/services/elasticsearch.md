# Elasticsearch

## Supported versions

- `7.6.1` as `elasticsearch76`
- `6.8.7` as `elasticsearch68`

## Kibana

- `7.6.1` as `kibana76`
- `6.8.7` as `kibana68`

## Data folder

Elastic search data lives in `elasticsearch/<version>/data` folder.

## Home folder

Home folder (`~`) lives in `elasticsearch/<version>/home` folder
